# Requirements: Glorb v1.2 — Focus Intelligence

**Defined:** 2026-04-18
**Core Value:** The timer counts down reliably and the app stays out of the way until needed.

## v1.2 Requirements

### Onboarding

- [ ] **ONBOARD-01**: App shows onboarding window on startup if user has no profile (name + ADHD status unset)
- [ ] **ONBOARD-02**: Onboarding step 1 — full-screen centered window with Glorb mascot, "Hi, I'm Glorbus Pallidus. You can call me Glorb.", breathing gradient background
- [ ] **ONBOARD-03**: Onboarding step 2 — "What should I call you?" name entry
- [ ] **ONBOARD-04**: Onboarding step 3 — ASRS 1.1 questionnaire intro text leads into 18 questions (one per screen), each with 5-dot Never→Very Often selector; Back button on each question
- [ ] **ONBOARD-05**: After questionnaire, ADHD diagnosis logic runs (per ASRS 1.1 scoring) and persists `hasADHD` boolean to store
- [ ] **ONBOARD-06**: "Retake Test" in settings panel relaunches the onboarding questionnaire flow

### Intervention Engine

- [ ] **INTERV-01**: `driftDetected()` increments a drift counter; `refocusDetected()` resets it to 0 and cancels all active escalation timers
- [ ] **INTERV-02**: After any non-terminal escalation event, `refocusDetected()` sends a "Focus regained." push notification
- [ ] **INTERV-03**: Weak × Regular path: 30s → push notif + 1 chime; every 10s → 2 chimes, then 3 chimes + "last reminder"; then terminate (open window, end timer, popup "Ready to continue focusing?")
- [ ] **INTERV-04**: Weak × ADHD path: 10s → push notif + 1 note; every 5s up to 5 pings with increasing notes, then 10s constant chime; terminate (open window, end timer, popup "You lost focus.")
- [ ] **INTERV-05**: Strong × Regular path: 15s → push notif + 1 chime, then every 10s (2 chimes, 3 chimes + last reminder); 2s full-screen Glorb flash "Still there?"; audio fades over 30s; vignette dims/brightens 60s; terminate (full black/white screen, big Glorb, "Focus.", 5s dwell to dismiss)
- [ ] **INTERV-06**: Strong × ADHD path: 10s → push notif, every 5s up to 5 pings; 5s full-screen Glorb flash "Still there?"; audio fades 30s; vignette 60s; same terminate screen as Strong × Regular

### CLI Simulator

- [ ] **CLI-01**: `node simulate.js drift` sends a drift signal to the running Electron app
- [ ] **CLI-02**: `node simulate.js refocus` sends a refocus signal to the running Electron app

### Wiring

- [ ] **WIRE-01**: Strength "Weak" routes `driftDetected()` to the Weak intervention path
- [ ] **WIRE-02**: Strength "Strong" routes `driftDetected()` to the Strong intervention path
- [ ] **WIRE-03**: `hasADHD: true` in store selects ADHD variant; `false` selects Regular variant

## Deferred Requirements

- **APP-01**: Break timer (5-min / 15-min) after work session
- **APP-02**: Session count tracker (4 pomodoros = long break)
- **APP-03**: Configurable timer durations via settings panel
- **APP-04**: Focus history persisted across app restarts
- **APP-05**: Strength setting "Auto" — auto-selects Weak/Strong based on ADHD profile (deferred per user decision)
- **DETECT-01**: Eye-tracking drift detection (eye-tracking not yet implemented)
- **DETECT-02**: App-tracking drift detection (foreground app monitoring not yet implemented)

## Out of Scope

| Feature | Reason |
|---------|--------|
| "Auto" strength mode | Deferred — Weak/Strong wired first per user decision |
| Eye-tracking / app-tracking | Not yet implemented — CLI simulator replaces for v1.2 |
| Animated questionnaire transitions | Nice-to-have; not in spec |
| Multi-language support | Not in scope |

|-----------|----------|-----------|-------------|-----------------|
| T-06-01 | Tampering | ipcMain.handle('send-serial') | accept | cmd values are controlled by app-owned renderer code; contextIsolation=true prevents external JS injection; only two valid values used ('SMILE\n', 'DEFAULT\n'); no user-supplied input reaches this handler |
| T-06-02 | Denial of Service | serialPort.write(cmd) | accept | Arduino firmware ignores unrecognized commands; write rate is bounded by timer events (max 1/5s); no loop or burst possible from this handler |
| T-06-03 | Information Disclosure | serial channel | accept | serial output goes only to Arduino over USB; no data read back from Arduino in this phase; no PII involved |
</threat_model>

**App configuration (SHELL-01):**
- `app.dock.hide()` — removes Dock icon on macOS
- `app.setActivationPolicy('accessory')` — prevents Dock entry even on first launch
- Wrap in `app.whenReady()` before creating Tray/window

**Tray icon (SHELL-01, D-08):**
- Load `glorb.png` using `nativeImage.createFromPath(path.join(__dirname, 'glorb.png')).resize({ width: 18, height: 18 })`
- Call `trayIcon.setTemplateImage(true)` to allow macOS dark/light menu bar inversion
- Create `new Tray(trayIcon)` and assign to module-level `let tray`

**BrowserWindow (SHELL-03, SHELL-04):**
- Create with these exact options:
  ```js
  new BrowserWindow({
    width: 220,
    height: 360,
    show: false,
    frame: false,
    resizable: false,
    alwaysOnTop: true,
    skipTaskbar: true,
    webPreferences: {
      preload: path.join(__dirname, 'preload.js'),
      contextIsolation: true,
      nodeIntegration: false
    }
  })
  ```
- `win.loadFile('renderer.html')`
- Do NOT call `win.show()` on startup — window starts hidden

**Window positioning (D-07):**
- In the tray click handler, get tray icon bounds: `const bounds = tray.getBounds()`
- Position the window: `win.setPosition(Math.round(bounds.x + bounds.width / 2 - 110), Math.round(bounds.y + bounds.height))`
- This centers the 220px window horizontally under the tray icon

**Tray click toggle (SHELL-02, D-02):**
```js
tray.on('click', () => {
  if (win.isVisible()) {
    win.hide()
  } else {
    const bounds = tray.getBounds()
    win.setPosition(
      Math.round(bounds.x + bounds.width / 2 - 110),
      Math.round(bounds.y + bounds.height)
    )
    win.show()
    win.focus()
  }
})
```

**Blur to hide (D-01):**
```js
win.on('blur', () => {
  win.hide()
})
```

**Cmd+Q quit (D-05):**
```js
const { globalShortcut } = require('electron')
app.whenReady().then(() => {
  globalShortcut.register('Command+Q', () => {
    app.quit()
  })
})
```

**IPC handler for renderer quit (D-04):**
```js
const { ipcMain } = require('electron')
ipcMain.handle('quit-app', () => {
  app.quit()
})
```

**Prevent default quit (keeps app running when window is closed):**
```js
app.on('window-all-closed', (e) => {
  e.preventDefault()
})
```

  <action>
Create `preload.js` at the project root. This is the only bridge between the renderer (untrusted) and the main process (trusted). Keep it minimal — expose only what the renderer needs.

```js
const { contextBridge, ipcRenderer } = require('electron')

contextBridge.exposeInMainWorld('glorb', {
  quit: () => ipcRenderer.invoke('quit-app')
})
```

This exposes `window.glorb.quit()` to the renderer. The renderer calls this when the user confirms quit in the overlay.

Do NOT expose `ipcRenderer` directly. Do NOT expose `require` or `shell`. Only expose the specific named function needed.

The channel name `'quit-app'` must match exactly what `ipcMain.handle('quit-app', ...)` registers in main.js.

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| renderer → main (IPC) | Renderer HTML/JS is untrusted; crosses to privileged main process via preload bridge |
| tray icon file → nativeImage | File path constructed from __dirname; must not be user-controlled |
| globalShortcut | Registers system-wide key hook; minimal surface (quit only) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Elevation of Privilege | preload.js IPC bridge | mitigate | Use `contextBridge.exposeInMainWorld` — never expose raw `ipcRenderer`. Only expose named functions with specific channel names. Verify in acceptance criteria. |
| T-01-02 | Elevation of Privilege | BrowserWindow webPreferences | mitigate | Set `contextIsolation: true` and `nodeIntegration: false` in BrowserWindow options. Verified in Task 2 acceptance criteria. |
| T-01-03 | Tampering | renderer.html loaded via loadFile | mitigate | Use `loadFile()` (local file) not `loadURL()` (remote). No remote content in Phase 1. |
| T-01-04 | Denial of Service | globalShortcut Command+Q | accept | Shortcut is standard macOS quit behavior. No sensitive data at risk. User intentionally invokes it. |
| T-01-05 | Information Disclosure | tray icon path via __dirname | accept | Path is hardcoded relative to app bundle directory. No user-controlled input in path construction. |
| T-01-06 | Repudiation | IPC quit-app channel | accept | Single-purpose channel (quit). No audit trail needed for a quit action on a single-user desktop app. |
</threat_model>

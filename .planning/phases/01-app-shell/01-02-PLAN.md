**Structure:**
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="Content-Security-Policy" content="default-src 'self'; script-src 'self' 'unsafe-inline'; style-src 'self' 'unsafe-inline'">
  <meta name="viewport" content="width=220, initial-scale=1.0">
  <title>Glorb</title>
  <link rel="stylesheet" href="renderer.css">
</head>
<body>
  <div id="app">
    <!-- Close button (× — D-03, D-04) -->
    <button id="btn-close" title="Quit Glorb">×</button>

    <!-- Quit confirmation overlay (D-04) -->
    <div id="quit-overlay" role="dialog" aria-modal="true" aria-labelledby="overlay-heading">
      <h2 id="overlay-heading">Quit Glorb?</h2>
      <p>This will close the app, not just hide the window.</p>
      <div class="overlay-buttons">
        <button id="btn-quit">Quit Glorb</button>
        <button id="btn-keep">Keep Running</button>
      </div>
    </div>

    <!-- Phase 2 timer UI will be injected here -->
  </div>

  <script>
    const btnClose = document.getElementById('btn-close')
    const btnQuit = document.getElementById('btn-quit')
    const btnKeep = document.getElementById('btn-keep')
    const overlay = document.getElementById('quit-overlay')

    function showOverlay() {
      overlay.classList.add('visible')
      btnQuit.focus()
    }

    function hideOverlay() {
      overlay.classList.remove('visible')
    }

    btnClose.addEventListener('click', showOverlay)

    btnQuit.addEventListener('click', () => {
      window.glorb.quit()
    })

    btnKeep.addEventListener('click', hideOverlay)
  </script>
</body>
</html>
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| renderer HTML/JS → preload bridge | Renderer calls window.glorb.quit() — crosses to main process via IPC |
| CSP → inline script | Renderer uses inline script; CSP must permit it safely |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Elevation of Privilege | renderer.html inline script | mitigate | Content-Security-Policy meta tag set to `default-src 'self'; script-src 'self' 'unsafe-inline'`. No external scripts. Renderer only accesses window.glorb (contextBridge surface), not raw Node APIs. |
| T-02-02 | Tampering | window.glorb.quit() invocation | accept | Only triggered by explicit user click on "Quit Glorb" button. No automated or programmatic trigger path. Single-user desktop app — no remote attacker surface. |
| T-02-03 | Information Disclosure | renderer HTML contents | accept | All content is static local HTML with no secrets, tokens, or user data. No network requests in Phase 1. |
| T-02-04 | Spoofing | Quit overlay UX | mitigate | Overlay copy explicitly states "This will close the app, not just hide the window." Per D-04, this prevents user confusion between hide and quit. Not a security threat per se but a UX integrity concern. |
| T-02-05 | Denial of Service | CSS user-select: none | accept | Prevents accidental text selection in the app window. Standard practice for desktop app chrome. No security concern. |
</threat_model>
